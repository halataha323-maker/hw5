# מסמך אפיון דרישות (PRD) - אוטומציית תוכן ל-MG4HOME

## מטרת הפרויקט
בניית מערכת מבוססת סוכני בינה מלאכותית (CrewAI) לייצור אוטומטי של מאמרי מוצר ותגיות SEO.

## דרישות המערכת
1. קבלת שם של מוצר כקלט.
2. **סוכן תחקירן:** איסוף נתונים טכניים ומפרטים על המוצר.
3. **סוכן כותב תוכן:** כתיבת מאמר שיווקי ומקצועי.
4. **סוכן SEO:** יצירת תגיות Title ו-Meta Description מותאמות למנועי חיפוש.
5. **סוכן QA:** בקרת איכות לתוכן ולתגיות לפני פרסום.

## כלים וטכנולוגיות
- Python
- רשת סוכנים של CrewAI
