# Product Requirements Document (PRD) - MG4HOME SEO Content Pipeline

## 1. מטרת המערכת
פיתוח פייפליין אוטומטי מבוסס סוכני בינה מלאכותית (באמצעות ספריית CrewAI) ליצירת תוכן שיווקי ותגיות מטא (SEO) עבור פריטי הציוד באתר MG4HOME. המערכת מקבלת שם מוצר ומחזירה מאמר מוכן לפרסום יחד עם תגיות Title ו-Meta Description.

## 2. הגדרת הסוכנים (Agents)
המערכת מורכבת מ-4 סוכנים הפועלים בתצורת עבודה סדרתית (Sequential Process):
1. **Market Researcher (תחקירן):** חיפוש ואיסוף מידע טכני, מפרטים ומגמות צרכנים על המוצר המבוקש.
2. **Content Writer (כותב תוכן):** כתיבת מאמר מוצר מקצועי ושיווקי (כ-300 מילים) על בסיס נתוני התחקירן.
3. **SEO Specialist (מומחה קידום אתרים):** גזירת תגיות מדויקות (Title עד 60 תווים, Description עד 155 תווים) מתוך המאמר.
4. **Quality Reviewer (בקר איכות):** עריכה סופית, בדיקת עובדות ווידוא כי התוכן תואם את המותג ומוכן לפרסום.

## 3. דרישות טכניות ומגבלות
* פיתוח בפייתון מבוסס CrewAI.
* שימוש ב-Agentic Tools: סוכן המחקר יקבל גישה לכלי חיפוש (כדוגמת SerperDevTool) כדי לאסוף מידע עדכני.
* שקיפות הרצה: חובה להגדיר `verbose=True` ברמת הצוות (Crew) והסוכנים כדי להציג את שלבי ה"מחשבה" והלוגים במסוף, לצורך צילומי המסך שיוגשו בדוח ה-README.
